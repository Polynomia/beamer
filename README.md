# Collection of `markdown -> Beamer` Templates for Pandoc

## compile

```bash
pandoc -t beamer --template themes/ponde.tex -s -o sample/sample.tex sample/sample.md
$TEX sample/sample.tex
```

see `sample/*.pdf`
